  • vxdg -g testdg adddisk testdg02=sdg - Ajouter un disque dans un diskgroup

Port Fonction Description
a GAB to LLT Global Atomic Broadcast or Node membership
b GAB to IOFENCE SCSI 3 Persistent Reservations - FENCING membership
d GAB to ODM (Oracle Disk Manager) Oracle Disk Manager Global Database Storage Locking
f GAB to CFS (Cluster File System) Cluster File System Lock and Metadata Traffic
h GAB to HAD High Availability Daemon Communication
o GAB to VCSMM driver System Kernel Generic Interface membership within Oracle
q GAB to QuickLog Daemon Intent Log Coordination
u GAB to CVM Temporary, used for node serialization during CVM join
v GAB to CVM Used for node serialization during CVM join
w GAB to vxconfigd in CVM Introduced in 3.5 to replace TCP/IP requirements for vxconfigd data transfer over Private Network
