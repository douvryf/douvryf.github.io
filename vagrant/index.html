Commandes

  • vagrant version - Afficher la version actuelle et la dernière de Vagrant

  • vagrant autocomplete install - Installer l'autocompletion

  • vagrant plugin list - Afficher les plugins installés

  • vagrant plugin install plugin_name - Installer un plugin

  • vagrant cloud search windows - Chercher une box sur le cloud

  • vagrant cloud box show gusztavvargadr/windows-10 - Afficher des infos sur une box

  • vagrant box add gusztavvargadr/windows-10 - Downloader une box

  • vagrant box list - Lister les boxes disponibles en local

  • vagrant init - Initialise en nouvel environnement vagrant en créant un Vagrantfile

  • vagrant validate - Valider un Vagrantfile

  • vagrant status - Afficher le statut de la machine vagrant. Statut: not created, poweroff, running

  • vagrant global-status - Afficher le statut de tous les environnements vagrant

  • vagrant up - Démarrer et provisionner la machine

  • vagrant halt - Arrêter la machine
  • vagrant destroy - Supprimer la machine
  • vagrant suspend - Suspendre la machine

  • vagrant resume - Réactiver la machine

  • vagrant port - Afficher le Port Forwarding pour la machine

  • vagrant ssh-config - Afficher le fichier ssh_config pour se connecter
  • vagrant ssh - Connexion à la machine

  • vagrant upload /var/tmp/srcfile /var/tmp/dstfile - Transférer un fichier dans la machine

  • vagrant snapshot list - Afficher les snapshots

  • vagrant snapshot save fd209 fd209-snapshot-avant-patch - Creer un snapshot
  • vagrant snapshot restore fd209 fd209-snapshot-avant-patch - Restorer un snapshot

  • vagrant snapshot delete fd209 fd209-snapshot-avant-patch - Supprimer un snapshot

  • vagrant test - Test vagrant
